# Conversion du fichier Markdown 

Sous Linux Debian, installez le package **pandoc**


    apt-get install pandoc

## Conversion en fichier LibreOffice


    pandoc Tercios_Brevis_Editio.fr.md -o Tercios_Brevis_Editio.fr.odt
